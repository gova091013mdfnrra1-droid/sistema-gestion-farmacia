# sistema-gestion-farmacia

## 1. Entrevista del Cliente
* **Desarrollador:** ¿Cuál es el objetivo principal del sistema para la farmacia?
* **Cliente:** Necesitamos registrar y controlar nuestro inventario de medicamentos (Nombre, Precio y Stock disponible) para atender rápido a los clientes.
* **Desarrollador:** ¿Requieren algún registro adicional de seguridad o auditoría?
* **Cliente:** Sí, nos gustaría un registro automático (log) cada vez que se dé de alta un nuevo medicamento para saber qué se añade al sistema y cuándo.
